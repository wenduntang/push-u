# 开源：极简 Agent 编排引擎，代码即编排，约 50 行核心

## 一句话

模型输出 Python 代码，引擎安全执行。无 JSON、无 DAG、无 LangChain。

## 特点

- 代码即编排：逻辑在代码里，不在 Prompt 里
- RestrictedPython + multiprocessing：安全 + 超时强杀
- 纯函数工具，`__doc__` 自动生成描述
- 三个 Demo：沙箱、编排引擎、符号神经融合

## 体验

```bash
pip install streamlit RestrictedPython
git clone https://github.com/wenduntang/push-u.git && cd push-u
streamlit run app.py
```
