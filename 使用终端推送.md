# 使用终端推送（无需双击脚本）

## 方式 1：push.sh（推荐）

```bash
cd ~/Desktop/沙箱/sandbox-security
./push.sh
```
按提示粘贴 Token，回车。

或直接传入 Token：
```bash
./push.sh ghp_你的token
```

## 方式 2：git push 手动输入

在终端中依次执行：

```bash
cd ~/Desktop/沙箱/sandbox-security
```

```bash
git remote remove origin 2>/dev/null
git remote add origin https://github.com/wenduntang/sandbox-security.git
```

```bash
git push -u origin main
```

出现提示时：
- **Username:** 输入 `wenduntang`
- **Password:** 粘贴你的 GitHub Token（不是密码）

---

**若提示权限错误，先执行：**
```bash
chmod +x 推送到GitHub.command
xattr -d com.apple.quarantine 推送到GitHub.command 2>/dev/null
```
